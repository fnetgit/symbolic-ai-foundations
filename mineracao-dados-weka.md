# Weka

Weka é uma coleção de algoritmos de aprendizado de máquina para tarefas de mineração de dados. Ele contém ferramentas para pré-processamento, classificação, regressão, clustering, regras de associação e seleção de atributos. O Weka é um software livre e de código aberto, desenvolvido na Universidade de Waikato, na Nova Zelândia.

## Mineração de dados com Weka

1. Obtendo o dataset (UCI repository, Kaggle, etc)
2. Estruturando o dataset(formato .arff)
3. Pré-processamento
4. Abrindo um dataset (carregando)
5. Visualizando os dados (Filtragem?)
6. Escolha uma tarefa/Análise de Resultados
   - Classificação
   - Regressão
   - Clustering
   - Associação
   - Seleção de Atributos
